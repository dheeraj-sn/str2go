str2go - String to Go Type Converter

Examples

Basic types:
Value: "42" | Type: int → 42
Value: "3.14" | Type: float64 → 3.14
Value: "true" | Type: bool → true
JSON arrays:
Value: '["a", "b", "c"]' | Type: []string → ["a", "b", "c"]
Value: '[1, 2, 3]' | Type: []int → [1, 2, 3]
Comma-separated values:
Value: "1,2,3,4,5" | Type: []int | Delimiter: "," → [1, 2, 3, 4, 5]
Value: "apple;banana;cherry" | Type: []string | Delimiter: ";" → ["apple", "banana", "cherry"]
JSON objects:
Value: '{"name":"John","age":30}' | Type: map[string]string → {"name":"John","age":"30"}
